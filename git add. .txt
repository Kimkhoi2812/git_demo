3.1.3 Branch(nhánh)
phát triển độc lập
tạo nhánh: git checkout -b tên của nhánh
chuyên nhánh lam viec: git checkout tên nhánh

13/4
- git merge <branch> :Merge <branch> into the current branch
đưa sự thay đổi từ nhánh này sang nhánh khác
- dùng để lấy code(pull) và đẩy (push) lên remote repository
-câu lệnh pull: git pull <remote repo> <branch name>
-câu lệnh git push <remote> <branch>
